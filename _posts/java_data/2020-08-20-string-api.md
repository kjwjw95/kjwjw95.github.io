이번 포스트는 자바에서 string에 관련된 method들을 모두 정리 하겠습니다.

* length
  - 설명 : string의 길이를 반환하는 메소드.
  - 리턴값 : string의 길이를 int로 리턴.
  - 예시
  ```java
  		test="abcd";
		System.out.println("test의 길이는 "+test.length());
  ```
  output : test의 길이는 4
